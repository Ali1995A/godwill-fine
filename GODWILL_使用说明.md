# Godwill 周易占卜工具使用说明

## 项目概述

Godwill是一个基于Python开发的周易64卦占卜工具，名字取"神意"之意。该工具实现了传统易经占卜术，既科学又简便，适合所有用户使用。

## 功能特点

- **完整64卦系统**：包含完整的周易64卦卦辞和详细解释
- **传统算法**：使用"混沌初开"传统占卜算法
- **简单易用**：只需简单的命令即可进行占卜
- **中文支持**：完整的中文卦辞解释

## 安装与使用

### 安装方法

1. 克隆仓库：
```bash
git clone https://github.com/Ali1995A/godwill.git
cd godwill
```

2. 安装项目：
```bash
python setup.py install
```

### 使用方法

#### 基本用法
```bash
# 随机占卜
gwill

# 输入特定数字进行占卜（最多6个数字）
gwill 1 2 3 4 5 6
```

#### 参数说明
- 支持输入1-6个整数
- 如果不输入参数，程序会自动随机生成数字
- 数字用于种子值，影响占卜结果

## 核心算法

项目实现了传统的"混沌初开"占卜算法：

1. **混沌初开**：将49颗算子分为天、地、人三部分
2. **三次变化**：通过3次变化过程生成最终卦象
3. **二进制编码**：使用6个爻位生成64卦的二进制编码

## 技术特点

- **Python 3.7+** 兼容
- **完整的卦辞数据库**：64个卦，每个卦都有详细的解释
- **命令行工具**：通过entry_points实现gwill命令
- **模块化设计**：清晰的代码结构，易于扩展

## 注意事项

1. 确保系统支持UTF-8编码以正确显示中文卦辞
2. 建议使用Python 3.7或更高版本
3. 卦辞内容基于传统易经解释，仅供娱乐参考
